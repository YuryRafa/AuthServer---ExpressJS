An authentication server built with Node.js and the Express.js framework, using JWT (JSON Web Tokens) for secure authentication and bcrypt for password encryption. The project aims to demonstrate a login, registration, and route protection system in a web application, integrating with a MySQL/MariaDB database and using dotenv to manage environment variables, all ensuring security.
